免费的"服务器": Github

第一篇博客呢，我想水一点，写一下在Github上"搭建网站"的过程。


首先，可以注意到，我们这个网址是jh-liang.github.io。我们要做的呢就是在GitHub上放上所有你需要用到的HTML页面、路径、档案，就跟你在其他服务器搭网站一样。


在Github上创建网站的方式和在Github上创建一个库的方式 完 全 一 样，是的是一模一样，从new repository到clone，所有步骤都完全一样，如果不清楚怎么创建库、怎么clone到本地的文件夹，可以去百度上找哦，很容易找到的。由于我不太清楚Mac和Linux是怎么样的，所以我在这里就不写了。


以下是几个需要注意的点：

  1. 在new repository命名库的时候，库的名字必须是 your_github_username.github.io (也就是你的网站名)，这样你的库就会自动变成一个网站
  2. 你的首页必须是index.html
  3. 在commit, push了至少一个html档案、你的首页之后，只要在浏览器输入 your_username.github.io 就能看到你的专属网站啦
  4. 你做网站需要的文件类型都可以放上去，css、png等等
  5. 但是，我估计通过Github搭建的网站要实现后台反馈收集资料的功能会比较难，我暂时还没试，尝试后会在本帖继续给大家分享。
